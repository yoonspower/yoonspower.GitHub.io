이용약관

최종 수정일: 2026-05-31

본 약관은 Recall ("앱") 의 이용 조건을 정합니다. 앱을 설치·사용함으로써 본 약관에 동의한 것으로 간주합니다.

1. 서비스 개요

Recall 은 Apple Watch·iPhone 으로 녹음한 음성을 텍스트로 전사하고 한국어로 요약해 주는 서비스입니다.

  • 녹음 음성은 개발자가 운영하는 서버로 전송되어 전사·요약 처리됩니다.
  • 전사·요약에는 OpenAI · Anthropic 의 API 가 사용되며, 자세한 데이터 처리는 개인정보 처리방침 을 참고하세요.

2. 무료 사용

  • 앱 다운로드는 무료입니다.
  • 가입(또는 첫 사용) 시점 기준 매 30일마다 무료 30분 의 녹음 처리 시간이 제공됩니다.
  • 무료 시간은 이월되지 않으며(소진 또는 30일 경과 시 30분으로 재충전), 광고는 없습니다.

3. 유료 시간 충전 (소비성 / Consumable)

무료 시간을 초과해 사용하려면 시간 팩을 구매할 수 있습니다. 모든 팩은 소비성(consumable) 상품으로, 구매한 시간은 만료되지 않습니다.

상품가격제공
1시간 팩₩1,900녹음 처리 60분
2시간 팩₩2,900녹음 처리 120분
3시간 팩₩3,900녹음 처리 180분 + 화자 구분 180분

3.1 충전 시간 사용

  • 구매한 시간은 계정(또는 기기) 의 잔액에 적립되며, 녹음을 처리할 때마다 사용한 분만큼 차감됩니다.
  • 3시간 팩에 포함된 화자 구분(speaker 분리) 시간은 별도 잔액으로 적립되며, 소진 시 일반 전사로 처리됩니다.
  • 충전 시간은 유효기간이 없습니다.

3.2 환불

  • 모든 결제는 Apple App Store 를 통해 이루어지며, 환불은 Apple 정책에 따릅니다 (reportaproblem.apple.com).
  • 소비성 상품 특성상, 이미 사용한 시간에 대해서는 환불이 제한될 수 있습니다.

3.3 미성년자 결제

  • 만 19세 미만 사용자가 결제를 진행할 경우 법정대리인(보호자) 의 동의가 필요합니다.
  • 법정대리인 동의 없이 이루어진 결제는 「민법」 제5조에 따라 취소 가능하며, Apple 환불 절차 (reportaproblem.apple.com) 를 통해 환불 신청할 수 있습니다.
  • 본 앱은 미성년자 여부를 직접 확인하지 않으며, Apple ID 의 가족 공유 및 구매 요청 기능 사용을 권장합니다.

3.4 결제 처리

  • 모든 결제는 Apple App Store (StoreKit 2) 를 통해 처리됩니다.
  • 개발자는 카드 정보 / 결제 정보를 수신하지 않습니다.
  • 구매 영수증은 서버에서 Apple 서명 검증을 거쳐 시간 적립에만 사용됩니다.

4. 사용자 책임

4.1 녹음 동의

  • 대화·회의를 녹음할 때, 녹음 대상자의 동의를 받는 것은 사용자 본인의 책임입니다.
  • 관련 법령(통신비밀보호법 등) 을 준수해야 하며, 무단 녹음으로 인한 법적 책임은 사용자에게 있습니다.

4.2 콘텐츠 책임

  • 녹음·전사·요약된 내용의 정확성은 보장되지 않습니다(AI 전사·요약 특성상 오류가 있을 수 있음).
  • 중요한 의사결정에 사용하기 전 원문을 확인하는 것을 권장합니다.

5. 면책 사항

  • 앱은 "있는 그대로" 제공됩니다.
  • 전사·요약의 부정확성, 처리 지연, 네트워크 단절, 데이터 손실로 인한 직간접 피해에 대해 개발자는 책임지지 않습니다. 다만, 개발자의 고의 또는 중대한 과실로 인한 손해, 그리고 「약관의 규제에 관한 법률」 등 강행법규에 의해 면책될 수 없는 책임은 본 조항의 적용에서 제외됩니다.

6. 서비스 변경 / 종료

  • 개발자는 앱 기능·가격·처리 방식을 사전 공지 후 변경할 수 있습니다.
  • 서비스 종료 시 최소 30일 전 앱 내 알림으로 공지하며, 미사용 충전 시간에 대한 처리 방안을 함께 안내합니다.

7. 지적 재산권

  • Recall 의 모든 코드·디자인·로고는 개발자에게 저작권이 있습니다.
  • 사용자가 녹음·생성한 콘텐츠의 권리는 사용자에게 있습니다.
  • 사용자에게는 비독점, 양도 불가, 취소 가능한 앱 사용권만 부여됩니다.

8. 준거법 및 분쟁 해결

  • 본 약관은 대한민국 법령에 따릅니다.
  • 분쟁이 발생할 경우 사용자의 주소지 관할 법원을 1심 법원으로 합니다.

9. 약관 변경

  • 본 약관은 사전 공지 후 변경될 수 있습니다.
  • 중대한 변경(가격 인상, 권한 축소 등) 은 변경 30일 전 앱 내 알림으로 공지합니다.
