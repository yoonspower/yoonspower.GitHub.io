개인정보 처리방침

최종 수정일: 2026-05-31

귀담아 (이하 "앱") 은 Apple Watch·iPhone 으로 녹음한 음성을 텍스트로 전사하고 한국어로 요약해 주는 서비스입니다. 이 방침은 앱이 어떤 정보를 수집하고, 음성 데이터가 어떻게 처리되는지를 투명하게 설명합니다.

핵심 요약: 녹음한 음성은 전사·요약을 위해 개발자가 운영하는 서버로 전송되어 처리됩니다. 전사에는 OpenAI 의 음성 인식 API, 요약에는 Anthropic 의 Claude API 가 사용됩니다. 처리 결과(전사문·요약)는 서버에 저장되어 사용자가 다시 열람할 수 있습니다. 원치 않으면 앱 내 "데이터 전체 삭제" 로 언제든 모두 지울 수 있습니다.

1. 수집·처리하는 정보

항목목적보관 위치

2. 음성 데이터 처리 흐름

  • 녹음한 음성은 암호화된 연결(HTTPS/TLS) 로 개발자가 운영하는 처리 서버로 전송됩니다.
  • 서버는 음성을 텍스트로 전사한 뒤 원본 음성 파일을 즉시 삭제합니다. 음성 원본은 영구 저장되지 않습니다.
  • 전사된 텍스트는 요약·분류·핵심 항목 추출을 위해 LLM(Claude) 으로 전달됩니다.
  • 최종 결과(전사문·요약·화자 구분)만 서버에 저장되어 앱에서 다시 열람할 수 있습니다.

3. 제3자 처리 위탁 (Sub-processors)

전사·요약 품질과 다중 사용자 처리를 위해 다음 외부 API 를 사용합니다. 각 사는 입력 데이터를 자사 정책에 따라 처리합니다.

업체전달 데이터용도정책
OpenAI녹음 음성음성 → 텍스트 전사 · 화자 구분openai.com/policies/privacy-policy
Anthropic전사 텍스트요약 · 분류 · 핵심 항목 추출anthropic.com/legal/privacy
Apple (StoreKit / APNs / Sign in with Apple)결제 영수증 · 푸시 토큰 · 계정 식별자결제 · 알림 · 로그인Apple 표준

API 제공사는 API 로 전달된 데이터를 자사 모델 학습에 사용하지 않는 정책(OpenAI/Anthropic 의 API 데이터 정책)을 따릅니다.

4. 수집하지 않는 정보

  • 연락처, 사진, 위치(사용자가 명시적으로 켜지 않는 한)
  • 광고 식별자 — 앱은 광고를 사용하지 않습니다
  • 제3자 분석·추적 SDK — 사용하지 않습니다

5. 결제 정보

  • 모든 결제는 Apple App Store 의 In-App Purchase (StoreKit 2) 로 처리됩니다.
  • 카드 정보는 Apple 이 직접 처리하며, 앱·개발자는 카드 정보를 수신하지 않습니다.
  • 앱은 구매 영수증의 트랜잭션 ID 만 받아 이용 시간 적립에 사용합니다.

6. 데이터 보존 및 삭제

  • 전사·요약 결과는 사용자가 삭제하기 전까지 서버에 보관됩니다.
  • 앱 → 설정 → "내 데이터 전체 삭제" 로 서버에 저장된 모든 녹음·전사·요약·계정 정보를 즉시 삭제할 수 있습니다 (되돌릴 수 없음).
  • 원본 음성 파일은 전사 처리 직후 자동 삭제되므로 별도 보존되지 않습니다.

7. 보안

  • 모든 통신은 TLS 로 암호화됩니다.
  • 처리 서버는 개발자가 직접 운영하며, 데이터는 본인 식별자로 격리되어 다른 사용자가 접근할 수 없습니다.

8. 아동 정보

앱은 만 4세 이상 사용 가능하나, 만 13세 미만 아동의 정보를 의도적으로 수집하지 않습니다. 음성 녹음 특성상 보호자 감독 하 사용을 권장합니다.

9. 변경 사항

이 방침이 변경될 경우, 변경된 방침이 적용된 앱 버전 출시 시점에 본 페이지의 "최종 수정일" 을 업데이트합니다.

English Summary (App Review reference)

Hearken records audio (via Apple Watch/iPhone), transcribes it to text, and summarizes it in Korean. Audio is sent over TLS to the developer-operated backend, transcribed via OpenAI's speech API, then deleted immediately (original audio is NOT retained). Transcripts/summaries are stored on the server (scoped to the user's anonymous device ID or Sign in with Apple account) until the user deletes them via Settings → "Delete all my data." Summarization uses Anthropic Claude (text only). No ads, no third-party analytics/tracking SDKs. Payments via Apple StoreKit (consumable time packs). Sub-processors: OpenAI (audio→text), Anthropic (text→summary), Apple (payments/push/sign-in). Contact: yoonspower@gmail.com.
