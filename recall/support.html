귀담아 사용 중 도움이 필요하시면 아래를 확인해 주세요.

자주 묻는 질문

녹음이 요약되지 않아요

1) 인터넷 연결을 확인하세요(녹음은 서버에서 전사·요약됩니다). 2) 남은 시간 잔액이 있는지 설정 → "남은 시간" 에서 확인하세요. 3) 잠시 후에도 결과가 안 오면 앱을 다시 열어주세요 — 처리 완료 시 자동으로 결과가 반영됩니다.

남은 시간이 부족해요

무료 30분은 30일마다 충전됩니다. 더 필요하면 설정 → "시간 충전" 에서 1·2·3시간 팩을 구매할 수 있습니다. 충전한 시간은 만료되지 않습니다.

화자 구분(누가 말했는지) 을 쓰고 싶어요

3시간 팩(₩3,900) 에 화자 구분 180분이 포함됩니다. 화자 구분 잔액이 있을 때 자동으로 적용되며, 소진되면 일반 전사로 처리됩니다.

구매한 시간이 반영 안 됐어요

인터넷 연결을 확인한 뒤, 설정 → "미완료 구매 복원" 을 눌러주세요. 결제는 됐는데 적립이 안 된 경우 자동으로 회수됩니다. 그래도 안 되면 이메일로 문의해 주세요.

내 녹음 데이터를 모두 지우고 싶어요

앱 → 설정 → "내 데이터 전체 삭제" 를 누르면 서버에 저장된 모든 녹음·전사·요약·계정 정보가 즉시 삭제됩니다(되돌릴 수 없음). 원본 음성은 전사 직후 자동 삭제되어 별도 보관되지 않습니다.

녹음 정확도가 떨어져요

조용한 환경, 마이크에 가까운 위치에서 녹음하면 정확도가 올라갑니다. 전사·요약은 AI 특성상 오류가 있을 수 있으니, 중요한 내용은 원문(전사) 을 함께 확인해 주세요.
