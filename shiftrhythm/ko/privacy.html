개인정보 처리방침

최종 수정일: 2026-06-03

ShiftRhythm (이하 "앱") 은 교대근무자의 생체시계(일주기리듬) 가이드를 제공합니다. 이 앱은 어떤 개인정보도 수집하거나 외부로 전송하지 않으며, 모든 계산은 사용자 기기 안에서만 이뤄집니다. 이 방침은 그 내용을 설명합니다.

1. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 사용자 계정 (앱에 로그인·회원가입 시스템 없음)
  • 위치, 연락처, 사진, 마이크 등 민감 권한
  • 광고 식별자·추적 정보 (앱에 광고·분석 SDK 없음)

2. 기기 내부에서만 처리되는 정보

앱은 다음 정보를 기기 내부에만 저장하며 (iOS SwiftData·앱 그룹 저장소), 외부로 전송하지 않습니다:

  • 사용자가 직접 입력한 근무 패턴·교대 출퇴근 시간·개별 날짜 보정
  • 직접 만든 사용자 정의 패턴
  • 앱 설정 (Apple 건강 연동 여부 등)
  • 생체시계 추정 캐시 (위상 계산 결과)

3. Apple 건강 (HealthKit) 데이터

사용자가 "Apple 건강 연동" 을 켜면, 앱은 추정 정확도를 높이기 위해 다음 데이터를 읽기 전용으로만 사용합니다:

데이터용도
수면 단계수면 점수·생체 정렬 추정
걸음 수 (활동)빛 노출 추정
심박변이도 (HRV)회복 신호 추정
안정시 심박회복 신호 추정
손목 체온컨디션 참고
  • 이 데이터는 기기를 떠나지 않습니다. 외부 서버로 전송하거나 제3자와 공유하지 않습니다.
  • 앱은 Apple 건강에 데이터를 쓰지 않습니다 (읽기 전용).
  • 연동은 선택 사항이며, 거부하거나 끄면 앱은 근무표만으로 동작합니다.
  • 권한은 언제든 iOS 건강 앱 → 데이터 접근 및 기기 에서 변경할 수 있습니다.

4. 네트워크 사용

  • 앱은 개발자 서버나 외부 서비스와 통신하지 않습니다 (계정·서버 없음).
  • iPhone 과 Apple Watch 사이의 데이터 동기화는 Apple 의 WatchConnectivity 를 통해 사용자의 두 기기 사이에서만 이뤄지며, 외부를 거치지 않습니다.

5. 제3자 SDK 및 광고

앱은 광고, 분석, 추적 SDK 를 포함하지 않습니다. 제3자에게 전송되는 데이터가 없습니다.

6. 데이터 보존 및 삭제

  • 모든 데이터는 기기 내부에만 저장됩니다 (앱 삭제 시 동시 삭제).
  • 앱 안에서 직접 삭제하려면: 앱 → 설정 → "모든 데이터 삭제".
  • Apple 건강 데이터는 앱이 보관하지 않으므로, 연동을 끄면 더 이상 읽지 않습니다.

7. 아동 정보

앱은 만 4세 이상 사용 가능합니다. 만 13세 미만 아동의 정보를 의도적으로 수집하지 않습니다.

8. 의료 고지

이 앱은 생활 리듬 관리를 돕는 참고용 정보를 제공하며, 의료기기나 진단 도구가 아닙니다. 질병의 진단·치료·예방을 목적으로 하지 않습니다. 건강 문제는 전문가와 상담하세요.

9. 변경 사항

이 방침이 변경될 경우, 변경된 방침이 적용된 앱 버전 출시 시점에 본 페이지의 "최종 수정일" 을 업데이트합니다.

English Summary (App Review reference)

ShiftRhythm is an on-device circadian-rhythm guidance app for shift workers. It does NOT collect personal information, does NOT use accounts, and has NO developer server. All computation happens on the user's device. With the user's permission it reads Apple Health data (sleep, steps, HRV, resting heart rate, wrist temperature) READ-ONLY to refine estimates; this data never leaves the device and is never shared with third parties. There are no advertising, analytics, or tracking SDKs. iPhone↔Apple Watch sync uses Apple WatchConnectivity between the user's own devices only. The app is a general-wellness reference tool, not a medical device.
