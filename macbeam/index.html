왜 MacBeam?

⚡

60fps 저지연 미러링

VideoToolbox H.264 하드웨어 인코딩. 네트워크 약하면 자동으로 fps/bitrate 조절.

⌨️

한글·영문·이모지 전부

한글 IME 직접 동기화. ⌘ ⌥ ⌃ ⇧ 단축키 스티키 모드. 자주 쓰는 단축키 10개 popup 저장.

🔊

시스템 오디오

Mac 의 모든 소리가 폰으로. 세션 중에는 Mac 스피커 자동 음소거, 종료 시 자동 복원.

🔒

P2P + TLS 1.2+

외부 중계 서버 없음. TLS 인증서 핀고정 (TOFU) 으로 MITM 차단. 32B PSK + HMAC.

👆

트랙패드 제스처

1손가락 = 좌클릭, 2손가락 탭 = 우클릭, 2손가락 스크롤, 핀치 줌. 더블탭 + 홀드 = 드래그.

🌐

VPN 호환

Tailscale, WireGuard, ZeroTier — 어디서든 두 기기가 같은 가상망에 있으면 작동.

자주 묻는 질문

Mac 서버는 왜 따로 설치해야 하나요?

점프데스크탑 / Screens / Splashtop 처럼 화면을 보낼 Mac 쪽에 작은 헬퍼 앱이 필요합니다. 무료 + 오픈 소스이고, 한 번만 설치하면 메뉴바에 자물쇠+방패 아이콘만 남아 보이지 않게 동작합니다.

외부 서버를 거치나요?

아니요. iPhone ↔ Mac 사이에 TLS 1.2+ P2P 연결만 합니다. 외부 중계 서버는 없으며, 데이터는 사용자 네트워크 밖으로 나가지 않습니다.

집 밖에서도 쓸 수 있나요?

네 — Tailscale, WireGuard, ZeroTier 같은 VPN 으로 두 기기를 연결하면 어디서든 작동합니다. Tailscale 은 무료 + 5분 셋업.

VNC 와 뭐가 다른가요?

macOS 기본 VNC 는 압축 없는 비트맵 — 같은 LAN 에서도 느림. MacBeam 은 H.264 하드웨어 인코딩으로 60fps 저지연 + 시스템 오디오 + 한글 IME 동기 + 우리만의 단축키/제스처 popup.

구독 취소는?

iPhone 설정 → Apple ID → 구독 → MacBeam → 취소. 다음 결제일 24시간 전까지. 현재 주기 종료까지는 사용 가능.
