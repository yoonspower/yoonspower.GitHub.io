개인정보 처리방침

최종 수정일: 2026-05-25

MacBeam (이하 "앱") 은 사용자의 개인정보를 소중히 다룹니다. 이 방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지를 설명합니다.

1. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 사용자 계정 (앱에 로그인 시스템 없음)
  • 위치, 연락처, 사진, 마이크 등 민감 권한
  • 사용자가 원격 제어하는 Mac 의 화면·소리·키 입력 — 모두 사용자 자신의 두 기기 사이에서만 직접(P2P) 전송되며, 개발자 서버를 거치지 않습니다.

2. 기기 내부에서만 처리되는 정보

앱은 다음 정보를 기기 내부 (iOS Keychain·UserDefaults) 에만 저장하며, 외부로 전송하지 않습니다:

  • 페어링한 Mac 의 호스트·포트
  • 페어링 후 발급된 사전 공유 키 (PSK, 32 bytes)
  • TLS 인증서 지문 (TOFU 핀고정)
  • 앱 설정 (해상도·FPS 등)
  • 광고 노출 cooldown 타임스탬프

3. 광고 및 추적

무료 사용 구간에 한해 Google AdMob 의 전면 광고가 표시됩니다.

  • 광고 SDK: Google Mobile Ads SDK
  • 수집 정보: 광고 식별자 (IDFA — App Tracking Transparency 동의 시), 기기 모델, OS 버전, 광고 노출/클릭 이벤트
  • 전송 대상: googleads.g.doubleclick.net, googlesyndication.com
  • ATT 동의 거부 시: 비개인화 광고만 표시되며, 광고 식별자는 전송되지 않습니다
  • 광고 영구 제거: 월정액 (₩1,900) 또는 평생 이용 (₩10,000) 결제 시

자세한 Google AdMob 정책: policies.google.com/technologies/ads

4. 결제 정보

  • App Store 의 In-App Purchase 만 사용합니다 (StoreKit 2)
  • 결제 처리 및 카드 정보는 Apple 이 직접 처리하며, 앱·개발자는 카드 정보를 수신하지 않습니다
  • 앱은 구매 영수증 (트랜잭션 ID + 만료일) 만 StoreKit 으로부터 받아 광고 제거 권한을 판정합니다
  • 영수증은 기기 내부에서만 검증되며 외부로 전송되지 않습니다

5. 네트워크 사용

  • 앱은 같은 Wi-Fi 네트워크 또는 사용자가 직접 구성한 VPN (Tailscale, WireGuard 등) 상의 Mac 과 직접 통신합니다
  • 통신은 TLS 1.2+ 로 암호화되며, 첫 페어링 시 인증서 지문을 핀고정합니다 (MITM 공격 차단)
  • 미디어 데이터 (화면·소리·키 입력) 는 두 기기 사이에서만 전송되며, 외부 서버를 거치지 않습니다

7. 데이터 보존 및 삭제

  • 모든 데이터는 기기 내부에만 저장됩니다 (앱 삭제 시 동시 삭제)
  • 페어링 정보·PSK·인증서 지문을 수동 삭제하려면: 앱 → 설정 → "페어링 초기화"

8. 아동 정보

앱은 만 4세 이상 사용 가능합니다. 만 13세 미만 아동의 정보를 의도적으로 수집하지 않습니다.

9. 변경 사항

이 방침이 변경될 경우, 변경된 방침이 적용된 앱 버전 출시 시점에 본 페이지의 "최종 수정일" 을 업데이트합니다.

English Summary (App Review reference)

MacBeam is an iOS↔Mac peer-to-peer remote desktop client. It does NOT collect personal information, does NOT use accounts, and does NOT route user media through any developer-controlled server. All streaming is direct between the user's own iOS device and Mac over TLS 1.2+ with TOFU certificate pinning. The only external network calls are to Google AdMob (for free-tier users who do not purchase the IAP). Privacy manifest (PrivacyInfo.xcprivacy) declares all data types and API access reasons per Apple policy.
