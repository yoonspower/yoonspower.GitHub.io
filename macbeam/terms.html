이용약관

최종 수정일: 2026-05-26

본 약관은 MacBeam ("앱") 의 이용 조건을 정합니다. 앱을 설치·사용함으로써 본 약관에 동의한 것으로 간주합니다.

1. 서비스 개요

MacBeam 은 사용자 본인의 iPhone/iPad 와 본인의 Mac 사이에서 화면·소리·입력을 직접 전달하는 원격 데스크탑 클라이언트입니다.

  • 두 기기가 같은 네트워크 또는 사용자가 직접 구성한 VPN 위에서 작동합니다.
  • 개발자가 운영하는 중계 서버는 없으며, 사용자 데이터는 외부 서버를 거치지 않습니다.

2. 무료 사용

  • 앱 다운로드는 무료입니다.
  • 최초 설치 후 8시간 무광고 로 사용 가능합니다.
  • 이후 광고 1회 시청당 추가 4시간 무광고 가 누적되며, 최대 24시간 까지 누적됩니다.

3.1 구독 (월정액)

  • 자동 갱신: 매월 같은 날짜에 Apple 계정으로 자동 결제됩니다.
  • 갱신 취소: 다음 결제일 24시간 전까지 iPhone 설정 → Apple ID → 구독 에서 취소.
  • 취소 시점: 현재 결제 주기 종료까지는 권한 유지됩니다.
  • 환불: Apple 정책에 따릅니다 (reportaproblem.apple.com).
  • 가격 변동: 가격이 변경되는 경우 변경 30일 전 앱 내 알림과 함께 사용자 동의를 받습니다. 미동의 시 다음 갱신 시 구독이 자동 해지됩니다.

3.2 평생 이용 (Non-Consumable)

  • 1회 결제로 영구 사용 권한이 부여됩니다.
  • 같은 Apple ID 의 모든 기기에서 자동 적용됩니다 (Restore Purchases 통해 복원).
  • 환불 정책은 Apple App Store 정책을 따릅니다.

3.3 미성년자 결제

  • 만 19세 미만 사용자가 결제를 진행할 경우 법정대리인 (보호자) 의 동의가 필요합니다.
  • 법정대리인 동의 없이 이루어진 결제는 「민법」 제5조에 따라 취소 가능하며, 14일 이내 Apple 환불 절차 (reportaproblem.apple.com) 를 통해 환불 신청할 수 있습니다.
  • 본 앱은 미성년자 여부를 직접 확인하지 않으며, Apple ID 의 가족 공유 및 구매 요청 기능 사용을 권장합니다.

3.4 결제 처리

  • 모든 결제는 Apple App Store (StoreKit) 를 통해 처리됩니다.
  • 개발자는 카드 정보 / 결제 정보를 수신하지 않습니다.
  • 결제 영수증은 기기 내부에서만 검증됩니다.

4. 광고

  • 무료 사용 구간에서 Google AdMob 의 전면(Interstitial) 광고가 표시됩니다.
  • 광고는 무광고 누적 시간이 0이 된 직후 연결 시점에만 표시됩니다.
  • 광고 트래커 동의 (App Tracking Transparency) 는 사용자가 거부할 수 있으며, 거부 시 비개인화 광고가 표시됩니다.
  • 자세한 정보: 개인정보 처리방침

5. 사용자 책임

5.1 본인 기기 한정 사용

  • MacBeam 은 사용자가 소유 또는 통제 권한이 있는 Mac 에만 접속하는 용도로 제공됩니다.
  • 타인의 Mac, 회사 정책상 금지된 기기, 또는 무단 접근 권한이 없는 기기에 접속하는 행위는 본 약관 위반이며, 해당 행위로 인한 법적 책임은 사용자 본인에게 있습니다.

5.2 네트워크 환경

  • 같은 Wi-Fi / VPN 환경이 필요합니다. 네트워크 설정·방화벽·VPN 구성은 사용자가 책임집니다.
  • 셀룰러 데이터로 사용할 경우 데이터 사용량은 사용자 통신사 요금제에 따릅니다.

5.3 보안

  • 페어링 PIN 은 1회용이며, 페어링 후 자동으로 32 byte 사전 공유 키 (PSK) 가 생성되어 두 기기의 키체인에 저장됩니다.
  • PSK 노출 시 즉시 Mac 메뉴바 → "PSK 재생성" 으로 변경하세요.
  • 신뢰할 수 없는 네트워크에서의 사용은 사용자 책임입니다.

6. 면책 사항

  • 앱은 "있는 그대로" 제공됩니다.
  • 사용 중 발생하는 데이터 손실, 화면 정지, 입력 누락, 네트워크 단절로 인한 직간접 피해에 대해 개발자는 책임지지 않습니다. 다만, 개발자의 고의 또는 중대한 과실로 인한 손해, 그리고 「약관의 규제에 관한 법률」 등 강행법규에 의해 면책될 수 없는 책임은 본 조항의 적용에서 제외됩니다.
  • Mac 의 화면이 외부에 노출될 가능성 (예: PIN 유출, PSK 유출, VPN 해킹) 은 사용자 보안 조치에 따라 달라지며, 개발자가 보장하지 않습니다.
  • 다음 작업 중 MacBeam 사용은 권장하지 않습니다:
    • 금융 거래, 본인 인증 등 고도의 보안이 필요한 작업
    • 의료, 항공, 원자력 등 인명·안전 직결 시스템 제어

7. 서비스 변경 / 종료

  • 개발자는 앱 기능, 가격, 광고 정책을 사전 공지 후 변경할 수 있습니다.
  • 서비스 종료 시 최소 30일 전 앱 내 알림으로 공지합니다. 평생 구매자에 대해서는 별도 처리 방안 (소스 코드 공개 등) 을 함께 안내합니다.

8. 지적 재산권

  • MacBeam 의 모든 코드·디자인·로고는 개발자에게 저작권이 있습니다.
  • 사용자에게는 비독점, 양도 불가, 취소 가능한 사용권만 부여됩니다.
  • 역공학, 디컴파일, 재배포는 금지됩니다 (단, GPL 라이선스 부분은 해당 라이선스를 따릅니다).

9. 준거법 및 분쟁 해결

  • 본 약관은 대한민국 법령에 따릅니다.
  • 분쟁이 발생할 경우 사용자의 주소지 관할 법원을 1심 법원으로 합니다.

10. 약관 변경

  • 본 약관은 사전 공지 후 변경될 수 있습니다.
  • 중대한 변경 (가격 인상, 권한 축소 등) 은 변경 30일 전 앱 내 알림으로 공지합니다.
  • 변경된 약관에 동의하지 않을 경우 앱을 삭제할 수 있으며, 환불은 Apple 정책에 따릅니다.
